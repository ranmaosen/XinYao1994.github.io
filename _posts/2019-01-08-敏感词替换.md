## 1：手机号部分替换为 * ##
11位手机号，保留前3后4，其余替换为*
    public static String maskPhoneNo(String str)
	{
		if (null == str || (str.trim().length() != 11))
		{
			return str; 
		} 
		return str.trim().replaceAll("(\\d{3})\\d{4}(\\d{4})", "$1****$2");
	}
